MSXπ固件(ファームウェア)
personal use only

MSXπは各種MSXカートリッジをエミュレートすることが出来るMSXカートリッジエミュレーターです。
現時点で以下の機能に対応しています。詳細は各ディレクトリ以下のreadme.txtを読んでください。

* ロム魂(256in1)
最大256のROMファイル(各マッパー対応)をMSXπに格納・起動する事が出来ます

* megaflash
メガROMカートリッジエミュレーター(FLASH版)
フラッシュメモリに格納したROMデータを使用するバージョン

* megausb
メガROMカートリッジエミュレーター(USB版)
USBメモリ上のROMファイル(ファイルサイズ128K以下)を使用するバージョン

* romflash
非メガROMカートリッジエミュレーター(FLASH版)
フラッシュメモリに格納したROMデータを使用するバージョン

* romusb
非メガROMカートリッジエミュレーター(USB版)
USBメモリ上のROMファイルを使用するバージョン

* pac
パナアミューズメントカートリッジエミュレーター

* new10x
コナミの新10倍カートリッジエミュレーター

* kanji
第一、第二水準漢字ROMカートリッジエミューレーター
HX-21内蔵ワープロソフトのみで動作確認(第二水準は未確認)

* ram
64K RAMカートリッジエミュレーター

==== 以下実験的対応 ====

* sunrise
Nextor対応Sunrise IDE(ハードディスク)エミュレーター

* mmapper
マッパーメモリー(240KB)エミュレーター

* disk
ディスクドライブエミュレーター

* keyboard
USBキーボードインターフェース
MSX/MSX2/MSX2+/MSXturboRで動くと思います。

* tape
データレコーダーエミュレーター(MSX/MSX2/MSX2+用)

* tape_r
データレコーダーエミュレーター(MSXturboR専用)
