仮想データレコーダー(テープ)装置のファームウェアです。
USBメモリ上のテープイメージファイルが使用出来ます。
MSXのみでの確認ですがMSX2/MSX2+でも動作すると思います。

ファームウェアについて
USB装置との相性が激しすぎるのでファームウェアを2種類用意しました。
必要い応じて使い分けてください。

tape.uf2: 標準バージョン
tape-BSTKH100BK.uf2: バッファロー製10キーBSTKH100BK対応バージョン


使用法

1. MSXに刺さっていない状態のMSXπをブートモードでPCに接続します。

MSXπのBOOTSELボタンを押しながらPCとUSB接続して下さい。
接続が成功するとドライブが認識されます。
HOSTジャンパの状態は不問です。

2. MSXπにtape.uf2ファイルを書き込みします。

1で認識されたドライブにtape.uf2ファイルをドラッグアンドドロップ(コピー)します。
コピーが完了するとドライブが見えなくなります。

3. 書込みが終わったらPCから外します。

注意：PCとの接続中はバスバッファの入力がフロート状態となります。
あまり良い状態ではないので長時間のPC接続は避けてください。

4. USBメモリのルートディレクトリに使用するテープイメージをコピーします。

USBメモリはFAT32でフォーマットして下さい。
ディスクラベルの無いUSBメモリ(直にFAT32フォーマットされた物)は問題があるかもしれません。

テープイメージのファイル名はtapeN.cas(N=0～9 例:tape0.cas, tape1.cas, ..., tape9.cas)にして下さい。
デフォルトで使用するイメージファイルはtape0.casです。tape0.casが存在しない場合はイジェクト状態で起動します。
ちなみに実装置とは異なり記録・再生の区別はありません。セーブを行えば記録・ロードを行えば再生が行われます。
読み取り専用のイメージファイルは書込み禁止のテープとして扱われます。

5. MSXπにUSBメモリを取り付けて完了

MSXπは一番若い番号の基本スロットに挿してください。
USB変換アダプター(OTGアダプター)が必要です。100円ショップ等で入手して下さい。
MSXπのHOSTジャンパをショートして下さい。

================

テープイメージ操作(イメージ交換・先頭への巻き戻し・末尾への早送り等)について
MSXπにUSBキーボードを接続する事によりテープイメージ操作が可能になります。
USBキーボードの接続にはUSBハブをご使用ください。

操作方法(当然ですが各キーはUSBキーボードのキーです)

* 0～9の数字キー(テンキー可)
値に対応するイメージファイルがマウントされます。※
例えば2を押すとtape2.casがマウントされます。該当ファイルが無い場合はイジェクト状態になります。

* リターンキー・エンターキー(テンキー可)
イメージファイルがイジェクトされます。※

* 左矢印キー又はテンキーの'-'
イメージ先頭への巻き戻し

* 右矢印キー又はテンキーの'+'
イメージ末尾への早送り

※テープ位置はイジェクト後もMSXπのメモリ上に保持されます。(位置情報は電源断・リセットで揮発します)

================

ご注意
USB機器に相性問題がかなり出やすいです。お気を付けください。
USBハブ、USBメモリ、USBキーボード以外のUSB装置を接続しないでください。
USBハブ、USBメモリ、USBキーボードを複数接続しないでください。
USB装置のホットスワップに対応していません。
